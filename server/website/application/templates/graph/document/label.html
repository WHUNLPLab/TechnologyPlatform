标注指南

此指南用于指导非结构化实体标注与非结构化关系标注的操作流程。在进行数据标注前，请详细阅读此指南。在标注过程中如遇到任何技术问题，可翻看此指南或者联系相关技术人员。

非结构化实体标注

下图为非结构化实体标注界面,其中，中部五彩按钮表示的是实体类型（图中为Person、Loc、Org等），按钮下方文本表示需要标注的文本。右上角为提交标注按钮，右下角为切换文本按钮，左下角是为自己的标注打分。

实体类型含义

  • Person （PER）：人物，如艺术家、政治家等人名
  • Loc （LOC）：地理位置，如国家、城市、地点
  • Org （ORG）：组织，如中国美术馆、中国政府、机构等
  • Work （WOR）：艺术作品，如蒙娜丽莎的微笑（不含书名号）等
  • Genre （GEN）：艺术流派，如印象派、抽象派等
  • Other （O）：所有不属于以上类型的文本类型

标注流程

  • 进入标注页面，查看文本。若无法判断实体类型，则直接切换下一条文本。若基本能判断，则进行下一步
  • 鼠标选中连续的需要标注的文本块，点击对应的实体类型按钮进行标注。重复此过程直到所有已知实体都进行了标注
  • 为自己的标注进行打分。系统将根据打分计算数据可靠程度
  • 若要修改标注，请用鼠标选中标注项（此时操作最小单位是一个标注项)，先标注为Other类型，再进行其他标注
  • 文本标注结束后，点击提交标注按钮，查看后台返回状态
  • 获取下一条标注数据，重复上述过程
