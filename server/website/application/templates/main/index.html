多轮对话

设计实现垂直领域的多轮对话系统，包括：（1）将非结构化自然语言命令转化为结构化的语义表示；（2）维护系统与用户交互的上下文信息。系统最终通过基于自然语言的交互，完成用户的命令

多意图

多意图识别计划在语音识别后的无标点文本中自动加入标点信息，划分句子边界，使得处理后的文本便于阅读理解和意图识别。该技术能提高许多下游任务的效率，如语义分析、机器翻译等。

更多

知识图谱

知识图谱面向中西方古典艺术，主要包括美术、音乐、戏曲、名著等。计划基于结构化、半结构化和非结构化文本，完成数据爬取解析、信息抽取以及知识融合建库等操作

更多

情感分析

对文本进行细粒度的情感分析，从而识别出文本所表达的情感及情感评价的对象或属性。该技术能够辅助消费者对产品进行选择，同时对企业收集消费者对产品的建议也有很大的帮助。

更多

短文本关键词抽取

利用词汇的统计信息和主题特征等抽取短文本关键词并根据重要性进行排序。该技术主要应用于短文本聚类、自动问答、自动文摘等领域。

更多

答案生成

答案生成能够利用领域问答语料及外部知识库为用户提问自动生成具有背景信息的描述性答案。该技术可用于智能客服、自动问答、信息检索等领域个性化答案的生成。

更多

阅读理解

基于阅读理解模型和复述生成模型，实现面向开放域的、生成式的智能问答系统。该系统能够对用户的各种问题描述做同义识别并给出问题答案。

更多

歌曲文案

通过理解歌曲语义和用户情感，自动在播放歌曲前后引入相关的自然语言描述文本。该技术通过与用户聊音乐的方式，加强与用户之间的沟通，提高用户听歌体验。

关于我们

项目组成员来自武汉大学计算机学院，长期专注于自然语言处理、信息检索、网络传播等方面的研究. 近五年来，主持了相关领域的中国国家自然科学基金项目3项、国家和省部市级科技计划项目共计12项.

近五年发表与本研究紧密相关的SCI/EI/ISTP/核心论文共计101篇，其中SCI25篇、CCF和通信领域C类以上会议论文19篇， 参编著作2部；授权国家发明专利和软件著作权7项；获得湖北省科技进步一等奖和自然科学二等奖各1项、国际会议最佳论文奖1项.
